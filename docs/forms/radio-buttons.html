これちょっとよく分からないな。説明として表示される "Drafts are〜" の方の <div> はclass="hidden peer-checked/draft:block" でもう片方が class="hidden peer-checked/published:block" なのだが、デフォルトで hidden が効くのは分かる、だが peer-checked/published:block で表示されるのがよく分からない。

→ 分かった hidden = display: none; で block = display: block; だから。

hidden はタグのステートと勘違いした。
